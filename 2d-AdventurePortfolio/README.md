# Hotel Portfolio

A 2D interactive portfolio game built with Kaboom.js

## Setup Instructions

1. Clone the repository:
```bash
   git clone <your-repo-url>
   cd hotel-portfolio
```

2. Install dependencies:
```bash
   npm install
```

3. Run the development server:
```bash
   npm run dev
```

4. Build for production:
```bash
   npm run build
```

## Tech Stack
- Kaboom.js
- Vite
- JavaScript
